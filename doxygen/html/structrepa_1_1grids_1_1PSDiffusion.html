Detailed Description

Diffusively load-balanced grid based on Diffusion class.

Ensures that the structure of the grid remains constant

The implementation prevents new neighbours from being added. But it is allowed to lose neighbors if no process gets a new neighbor (this is possible via the corners). These lost neighbours can be recovered in later operations.
